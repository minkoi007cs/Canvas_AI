How to get your Canvas API token:

  1. Go to Canvas → click your profile in the top right
  2. Select Settings
  3. Scroll down to "Approved Integrations"
  4. Click "New Access Token"
  5. Purpose: Canvas AI Helper
  6. Click "Generate Token"
  7. Copy the token (long string) and paste it below
💡 Tip: You can revoke this token anytime in Canvas settings. Your password is never stored on this app.

Token will be encrypted and stored securely.

🔒 Security: Your Canvas API token is encrypted with a secure key. We never store your password.
